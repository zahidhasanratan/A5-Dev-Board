Question-1: What are the different ways to select an element in the DOM?

There are various ways to select an element in the DOM. Some of them are getElementById, by which we can select an element by its unique ID, getElementsByClassName, by which we can get all elements with a specific class, getElementsByTagName, by which we can select elements by their tag name ETC.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML let us to get or change the HTML inside an element, it also includes HTML tags.
innerText gives us the visible text content inside an element.
textContent is similar to innerText, but it returns the raw text.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a JavaScript design pattern where we can attach a single event listener to a parent element instead of adding listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM

Question-5: How do you create, add, and remove elements using JavaScript?

To add, remove, and modify HTML elements using JavaScript, we can use methods like createElement , appendChild , removeChild , and properties like innerHTML and textContent.
